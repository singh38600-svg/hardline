# Roadmap

## v0.1 — Claude Code MVP

- typed policy schema;
- file-write, command, dependency, and completion policies;
- Claude Code PreToolUse, PostToolUse, and Stop hooks;
- deterministic runtime with zero LLM calls;
- local decision receipts and explanations;
- adversarial policy simulations.

## v0.2 — Portable policy layer

- Codex and Gemini CLI adapters;
- Git pre-commit and GitHub Actions adapters;
- task-scoped write boundaries;
- one-use policy exceptions;
- signed policy lockfile.

## v0.3 — Policy intelligence

- transcript-based policy suggestions;
- conflict and duplicate detection;
- AST and Semgrep plugins;
- reproducible soft-rule benchmark suite.
